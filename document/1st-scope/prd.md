# PRD - Todoアプリ

## 概要

| 項目 | 内容 |
|------|------|
| アプリ名 | Todoアプリ |
| プラットフォーム | iOS 18.0+ / iPhone のみ |
| 目的 | CodingAgent トレーニング用のTODO管理アプリ |

---

## ユーザーストーリー

### タスク管理

- ユーザーはタスクを作成・編集・削除できる
- ユーザーはタスクに タイトル・メモ・優先度 を設定できる
- ユーザーはタスクを完了状態にマークでき、一覧で「すべて」「未完了」のフィルターを切り替えられる
- ユーザーはタスクの3点メニューから編集・削除を実行できる
- フィルターの選択状態は次回起動時に復元される

---

## 機能一覧

### タスク

| 機能 | 説明 |
|------|------|
| 作成 | タイトル（必須）・メモ・優先度を設定して作成 |
| 編集 | 3点メニューから遷移。全項目を編集可能 |
| 削除 | 3点メニューからタスクを完全削除 |
| 完了 | チェックボックスで完了状態をトグル。完了済みは打ち消し線＋優先度タグがグレー表示 |
| フィルター | ボトムシートで「すべて」「未完了」を切り替え。選択状態はUserDefaultsに保存し、次回起動時に復元 |

### タスクのデータ項目

| 項目 | 型 | 必須 | 説明 |
|------|----|------|------|
| id | UUID | ○ | 一意識別子 |
| title | String | ○ | タスクタイトル |
| memo | String | - | メモ |
| priority | Enum | ○ | 高・中・低（デフォルト: 中） |
| isCompleted | Bool | ○ | 完了フラグ（デフォルト: false） |
| createdAt | Date | ○ | 作成日時 |
| updatedAt | Date | ○ | 更新日時 |

---

## 画面一覧

| 画面名 | 説明 |
|--------|------|
| タスク一覧 | タスクの一覧。フィルターボタンで「すべて」「未完了」を切り替え可能 |
| タスク作成 | 新規タスクの入力フォーム |
| タスク編集 | 既存タスクの編集フォーム。3点メニューから遷移 |
| フィルター選択 | ボトムシートで一覧のフィルターを選択 |

---

## 非機能要件

| 項目 | 内容 |
|------|------|
| データ永続化 | UserDefaults（JSONエンコード） |
| オフライン | 常時オフライン動作（外部API依存なし） |
| アーキテクチャ | MVVM + Repository パターン |
| 非同期処理 | async/await |
