# PRD - Todoアプリ

## 概要

| 項目 | 内容 |
|------|------|
| アプリ名 | Todoアプリ |
| プラットフォーム | iOS 18.0+ / iPhone のみ |
| 目的 | CodingAgent トレーニング用のTODO管理アプリ |

---

## ユーザーストーリー

### タスク管理

- ユーザーはタスクを作成・編集・削除できる
- ユーザーはタスクに タイトル・メモ・期限日・優先度 を設定できる
- ユーザーはタスクを完了状態にマークでき、一覧で完了済みタスクの表示・非表示を切り替えられる
- ユーザーは完了済みタスクを削除できる

### 写真添付（カメラ権限）

- ユーザーはタスクに写真を1枚添付できる
- カメラまたはフォトライブラリから選択できる
- カメラ権限が未許可の場合、権限リクエストを行う
- 権限が拒否された場合、設定アプリへ誘導する

### 現在地の紐づけ（位置情報権限）

- ユーザーはタスク作成・編集時に現在地を紐づけられる
- 位置情報権限が未許可の場合、権限リクエストを行う
- 権限が拒否された場合、設定アプリへ誘導する
- 紐づけた位置情報はタスク詳細に地名（住所）として表示する

## 機能一覧

### タスク

| 機能 | 説明 |
|------|------|
| 作成 | タイトル（必須）・メモ・期限日・優先度・写真・現在地を設定して作成 |
| 編集 | 全項目を編集可能 |
| 削除 | タスクを完全削除 |
| 完了 | 完了状態にトグル。一覧での表示・非表示を切り替え可能 |
| 削除（完了済み） | 完了済みタスクを削除 |

### タスクのデータ項目

| 項目 | 型 | 必須 | 説明 |
|------|----|------|------|
| id | UUID | ○ | 一意識別子 |
| title | String | ○ | タスクタイトル |
| memo | String | - | メモ |
| dueDate | Date | - | 期限日 |
| priority | Enum | ○ | 高・中・低（デフォルト: 中） |
| photoData | Data | - | 添付写真のバイナリ |
| location | Location | - | 緯度・経度・住所文字列 |
| isCompleted | Bool | ○ | 完了フラグ（デフォルト: false） |
| createdAt | Date | ○ | 作成日時 |
| updatedAt | Date | ○ | 更新日時 |

---

## 画面一覧

| 画面名 | 説明 |
|--------|------|
| タスク一覧 | タスクの一覧。完了済みタスクの表示・非表示を切り替え可能 |
| タスク作成 | 新規タスクの入力フォーム |
| タスク編集 | 既存タスクの編集フォーム |
| タスク詳細 | タスクの詳細表示 |

---

## 非機能要件

| 項目 | 内容 |
|------|------|
| データ永続化 | UserDefaults（JSONエンコード） |
| 権限ハンドリング | カメラ・位置情報。拒否時は設定アプリへ誘導 |
| オフライン | 常時オフライン動作（外部API依存なし） |
| アーキテクチャ | MVVM + Repository パターン |
| 非同期処理 | async/await |
