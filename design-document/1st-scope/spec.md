### 遷移図

```
タスク一覧
  ├── [ナビゲーションバー +ボタン] → タスク作成（モーダル）
  │     └── [保存 / キャンセル] → タスク一覧に戻る
  ├── [タスクの3点メニュー → 編集] → タスク編集（モーダル）
  │     └── [保存 / キャンセル] → タスク一覧に戻る
  ├── [タスクの3点メニュー → 削除] → 確認アラートを表示 → 削除してタスク一覧に戻る
  └── [フィルターボタン] → フィルター選択（ボトムシート）
        └── [選択 / 閉じる] → タスク一覧に戻る
```

## 2. データモデル定義

### タスク

| フィールド | 型 | 必須 | デフォルト | 説明 |
|------------|----|------|------------|------|
| id | UUID | ○ | 自動生成 | 一意識別子 |
| title | 文字列 | ○ | - | タスクタイトル |
| memo | 文字列 | - | 空文字 | メモ |
| priority | 優先度 | ○ | 中 | 高・中・低の3段階 |
| isCompleted | 真偽値 | ○ | false | 完了フラグ |
| createdAt | 日時 | ○ | 作成時刻 | 作成日時 |
| updatedAt | 日時 | ○ | 作成時刻 | 更新日時 |

### 優先度

| 値 | 表示ラベル |
|----|------------|
| high | 優先度 高 |
| medium | 優先度 中 |
| low | 優先度 低 |

### フィルター

| 値 | 表示ラベル | 挙動 |
|----|------------|------|
| all | すべて | 完了済み含む全タスクを表示 |
| incomplete | 未完了 | 未完了タスクのみ表示 |

### バリデーションルール

| フィールド | ルール |
|------------|--------|
| title | 空文字・空白のみ不可。最大30文字。無効な場合は保存を禁止する |
| memo | 最大100文字 |
| priority | 必ず1つ選択。未選択状態は作らない |
